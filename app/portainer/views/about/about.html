关于 Portainer
Portainer 是一个免费开源软件 给你 来自 portainer.io 和 贡献者。 欢迎加入交流、bug反馈QQ群

我们在 Portainer 提供免费和开源的软件。 为了了解将我们非常有限的开发预算用在何处最好，我们使用 rudimentary 分析。

Opt-out
  • 您可以在启动 Portainer 时通过将 --no-analytics 标志作为 docker run 命令的一部分来选择退出。
  • 如果您认为我们可以改进我们的分析方法，请务必让我们知道！ 有一个公开的讨论在 Github
我们收集什么 & GDPR
  • 我们不知道谁在使用 Portainer，它在哪里使用，它的使用规模如何，我们所知道的（从分析中）是 Portainer 的使用频率以及应用程序中的哪些页面 最常用。
  • 由于我们只收集极少量完全匿名的数据，因此认为不需要选择加入。
